## js原生函数

js常用的原生函数有：
- String()
- Number()
- Boolean()
- Array()
- Object()
- Function()
- RegExp()
- Date()
- Error()
- Symbol() (ES6语法)

### String原生函数的一些基础用法

```
let hll = new String("Hello world!")
console.log(hll)
console.log(hll.toString())
console.log(typeof hll)
console.log(hll instanceof String)

// 执行结果 
// String {"Hello world!"}
// Hello world!
// object
// true
```
